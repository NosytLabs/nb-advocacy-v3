# nb-advocacy-v3

  Public-interest research and advocacy archive on New Brunswick provincial issues. Built with Next.js.

  ## Scope

  Long-form articles covering:
  - Healthcare crisis (ER deaths, gender gap, dental care)
  - NB Power (rate hikes, Lepreau spending, emissions)
  - Corporate concentration (Irving forestry, media, tax extraction)
  - Drug policy (BC decriminalization results, Canadian policy 2026)
  - Housing, child poverty, food bank crisis, population decline
  - Government accountability (Holt lobbying, Dornan firing, ATCON)

  ## Stack

  Next.js (App Router), TypeScript, static article pages.

  ## Develop

  ```bash
  pnpm install
  pnpm dev
  ```
