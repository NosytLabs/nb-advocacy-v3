# nb-advocacy-v3

> Public-interest research and advocacy archive on New Brunswick provincial issues.

[![Live Site](https://img.shields.io/badge/Live-Site-6366f1?style=flat-square)](https://nosytlabs.github.io/nb-advocacy-v3)

[![TypeScript](https://img.shields.io/badge/TypeScript-3178C6?style=flat-square&logo=typescript&logoColor=white)](https://www.typescriptlang.org/)

A long-form journalism and research archive covering systemic issues in New Brunswick, Canada. Built with Next.js App Router for fast, SEO-optimized static delivery.

## Coverage Areas

| Topic | Issues Covered |
|---|---|
| **Healthcare** | ER deaths, gender gap in care, dental coverage |
| **Energy** | NB Power rate hikes, Lepreau spending, emissions |
| **Corporate Power** | Irving forestry, media consolidation, tax extraction |
| **Drug Policy** | BC decriminalization outcomes, Canadian policy 2026 |
| **Social Issues** | Housing, child poverty, food banks, population decline |
| **Accountability** | Government lobbying, whistleblower firings, ATCON |

## Stack

- [Next.js](https://nextjs.org/) (App Router)
- TypeScript
- Static export for GitHub Pages hosting

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000)

## Build

```bash
npm run build
```
